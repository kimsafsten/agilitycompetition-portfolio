# Features

## Public user features
- View upcoming competitions
- Submit competition registrations
- View published results and ranking

## Admin features
- Create and update competitions
- Manage class groups and registration settings
- Manage entries and start lists

## Secretary features
- Run competition-day workflow in a dedicated view
- Set runtime values and statuses
- Mark disqualifications and reruns
- Update start order when needed

## Technical features
- REST API with role-based authorization
- Validation and domain rules in backend services
- Strongly typed frontend API layer (TypeScript)
- Cloud deployment with separated frontend/backend/database
